[size=6][b]CC-Packer[/b][/size]

A simple, standalone tool to merge Fallout 4 Creation Club content into unified archives, reducing plugin count and improving load times.

[size=5][b]âœ¨ Features[/b][/size]

[list]
[*][b]Automatic Detection[/b]: Finds your Fallout 4 installation and Archive2.exe
[*][b]Smart Merging[/b]: Merges all cc*.ba2 files in your Data folder
[*][b]Crash Prevention[/b]: Automatically splits large texture archives to prevent issues
[*][b]BA2 Integrity Verification[/b]: Validates all created archives to detect corruption
[*][b]Separate Audio Archive[/b]: Sound files packed uncompressed to prevent audio issues
[*][b]Loose STRINGS Extraction[/b]: STRINGS files extracted to Data/Strings for reliable localization
[*][b]Merged File Detection[/b]: Detects previously merged files and prevents re-merging
[*][b]Comprehensive Error Handling[/b]: Detailed error messages instead of cryptic failures
[*][b]Administrator Detection[/b]: Warns if running in protected locations without admin rights
[*][b]Safety First[/b]: Backs up all original files to Data/CC_Backup before making changes
[*][b]Easy Restore[/b]: One-click restoration to original state
[*][b]Simple GUI[/b]: Clean, intuitive interface - no configuration needed
[/list]

[size=5][b]ğŸ’¡ Why Use CC-Packer?[/b][/size]

[b]The Problem[/b]
[list]
[*]Fallout 4 has a 255 plugin limit (ESM/ESP files)
[*]Each Creation Club item = 1 plugin slot
[*]With 170+ CC items now available, you could lose over half your plugin slots!
[*]Game performance degrades with hundreds of small BA2 archives (346+ files)
[/list]

[b]The Solution[/b]
[list]
[*]Merges all CC archives into just 3-4 unified files
[*]Reduces plugin count from 170+ to just 2-3 light master plugins
[*]Improves game loading times significantly
[*]Frees up plugin slots for your favorite mods
[*]Prevents texture and audio corruption through smart archive handling
[/list]
